# Coding Style

## Runtime

- Bun

## Language

- JavaScript
- ESM

## Rules

Gunakan:

- async/await
- const
- let jika perlu
- early return
- destructuring

Hindari:

- callback hell
- duplicate code
- nested if berlebihan
- global variable baru

## Imports

Gunakan alias project jika tersedia.

Jangan membuat utilitas baru jika sudah ada utilitas serupa.

## Error Handling

Selalu gunakan try/catch.

Error harus memiliki pesan yang jelas.

Jangan swallow error.
